AUTOLOAD HELPER

Sobre

Passo a passo para mostrar como configurar o autoload php composer

Requisitos

  • PHP
  • Gerenciador de Pacotes PHP (Composer)
  • Composer Autoload
  • Conhecimento em Namespace

Autoload

Configuração

Para recarregar/atualizar o ambiente com o composer é preciso editar corretamente o arquivo composer.json obedecendo a seguinte estrutura para definição de namespace: 

"{{{VENDOR}}}\\{{{NAMESPACE}}}\\": "src/"

Exemplo

"{{{VENDOR}}}\\Controllers\\": "controllers/"

Um modelo de configuração para autoload no arquivo composer.json pode ser visto a seguir:: 

"autoload": {
    "psr-4": {
        "{{{VENDOR}}}\\Controllers\\": "controllers/",
        "{{{VENDOR}}}\\Interfaces\\": "controllers/interfaces/",
        "{{{VENDOR}}}\\Services\\": "services/",
        "{{{VENDOR}}}\\Models\\": "models/",
        "{{{VENDOR}}}\\Models\\Interfaces\\": "models/interfaces/",
        "{{{VENDOR}}}\\Libraries\\": "libraries/"
    }
},

Executando

Para atualizar o autoload com os namespace inseridos no composer.json, use: 

composer dump-autoload

Exemplo de uso

<?php

namespace HunterCodexs\Controllers;

class HunterCodexsController
{
    public function __construct()
    {
        //Code Here ...
    }
}
