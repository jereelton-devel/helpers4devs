[ NPM HELPER ]

Criando/Publicando um modulo no NPM

- Criar uma conta no NPM
- Criar o projeto
- Executar npm init para gerar o package.json
	- Preencher o cadastro no terminal
- Criar um repostito no GitHUB (PUBLICO)
- Verificar usuario atual:
	-npm whoami
- Ficar atento no local onde esta o projeto, deve-se estar dentro da pasta onde ele esta
	- cd /home/usuario/projetoNPM
- Efetuar o login no NPM
	- npm login
		- usuario
		- senha
		- email
- Verificar o arquivo package.json antes de publicar
	- "respository": {"type": "git", "url": "git://github.com/vendor/projeto"}
- Publicar o projeto
	-npm publish
!IMPORTANTE: Se remover um pacote, tem que esperar 24horas para publicar um com o mesmo nome.
