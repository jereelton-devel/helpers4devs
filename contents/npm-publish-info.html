NPM HELPER

Sobre

Criar e gerenciar um modulo/pacote no repositório do NPM.

Requisitos

  • Ubuntu 18.04 ou superior
  • NPM instalado
  • Ter uma conta no NPM
  • Ter uma conta no GitHub
  • Ter um projeto no GitHub com visão Publica

Procedimentos Gerais

Criando um modulo para o NPM

- Criar uma conta no NPM (https://www.npmjs.com/)
- Criar o projeto em um diretorio qualquer da sua maquina ou clocar um projeto existente no seu GitHub
- Executar npm init para gerar o package.json dentro da pasta do projeto - Preencher o cadastro no terminal
- Criar um repostito no GitHUB (PUBLICO) caso ainda não tenha feito na etapa anterior
- Verificar usuario atual: 

	npm whoami

- Ficar atento no local onde esta o projeto, deve-se estar dentro da pasta onde ele esta 

	cd /home/usuario/projetoNPM

- Efetuar o login no NPM 

	npm login
		Informar usuario
		Informar senha
		Informar email

- Verificar o arquivo package.json antes de publicar 

	"respository": {"type": "git", "url": "git://github.com/vendor/projeto"}

Publicando o modulo no NPM

	npm publish

Removendo um pacote

	npm login
	npm unpublish --force yourPackage

NOTA: Se remover um pacote, pode ser necessário aguardar por cerca de 24 horas para publicar um modulo com o mesmo nome.
