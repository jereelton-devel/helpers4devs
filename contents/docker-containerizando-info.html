DOCKERISE PHP HELPER

Sobre

Criar um ambiente completo para desenvolvimento, contendo um ou mais containers, um container para banco de dados Mysql, PHP containerizado e NGINX.

Exemplo: https://github.com/jereelton-devel/dockerise

Requisitos

  • Ubuntu 18.04 ou superior
  • Usuario root (sudo)
  • Conhecimento em comandos Linux

Docker Container

Importante

- É possivel fazer o pull de imagens do docker como o php7-fpm ou usar um builder, como o contido no projeto citado no inicio desse documento, o builder é chamado de Dockerfile.

Preparando ambiente

- Criar estrutura de diretorios e arquivos
- Criar arquivo docker-compose.yml
- Confgurar arquivo docker-compose.yml
- Criar Network external se necessario (exemplo: docker network create dev_gateway => {6c87c75ff021f620a83f293fc60578db8c66adbdf167a329bb4593c5b1894f9b})
- Criar pastas do projeto e aplicações
- Executar o projeto: docker-compose up -d
