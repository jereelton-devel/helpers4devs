AWS ELASTIC IP

Sobre

Alocar uma endereço de IP elastico para ser usado por uma instancia, esse IP poderá ser reutilizado por outras instancias caso seja necessário remover alguma instancia e criar novamente sem perder o IP alocado anteriormente.

Requisitos

  • Ter uma conta criada e aprovada e ativa na AWS
  • Escolher a zona corretamente: nesse caso sa-east-1c
  • Ter uma instancia EC2 previamente criada

IP Elastico

Configuração

- Para iniciar a configuração do IP Elastico, acesse o painel EC2 no console da AWS e procure por Elastic IPs no menu lateral esquerdo da tela (Rede e Segurança).
- Ao acessar a tela de Endereços IP Elasticos você podera ver os IPs que já estão alocados e gerenciar, ou então adicionar um novo IP Elastico clicando no botão "Alocar endereço IP elástico" no canto direito superior da tela.
- Com o endereço de IP Elastico alocado é possíel associar e desassociar o mesmo a uma instancia EC2 criada no console da AWS.
